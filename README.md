## Installation

## How to launched it locally ?

### FrontEnd with mock :

Go to the cloned repository on your computer with bash :

```
$ cd JonathanChosson_2_15102010
```

install the package (describe in `package.json`) :

```
$ npm install
OR
$ yarn install
```

Start your local server :

```
$ yarn start
```
